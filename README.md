# Firebase Studio Next.js Starter

This is a starter project for building modern web applications in Firebase Studio. It comes pre-configured with a powerful and efficient tech stack to get you building right away.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (with App Router)
- **UI**: [React](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [ShadCN UI](https://ui.shadcn.com/)
- **AI**: [Google's Genkit](https://firebase.google.com/docs/genkit)

## Getting Started

To run the project locally, follow these steps:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

The application will be available at [http://localhost:9002](http://localhost:9002).

## Folder Structure Overview

- `src/app`: The core of your application, containing all pages and layouts. The file `src/app/page.tsx` is the homepage.
- `src/components/ui`: Contains all the reusable UI components from ShadCN.
- `src/ai`: Your Genkit AI flows and configuration live here.
- `src/lib`: Shared utility functions and helper code.
- `public`: For static assets like images or fonts that don't need to be processed.
